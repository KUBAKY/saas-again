#### 技术栈确认
**后端技术栈**:
- NestJS ^10.0.0 + TypeScript ^5.0.0

- TypeORM ^0.3.0 + JWT ^9.0.0
- Swagger ^7.0.0

**前端技术栈**:
- Vue 3 ^3.3.0 + TypeScript ^5.0.0
- Element Plus ^2.4.0 + Pinia ^2.1.0

**AI开发工具**:
