## **产品需求文档（PRD） - 健身房多品牌管理 SaaS 系统**

### **1. 项目背景**

本项目旨在为公司构建一套**多品牌、多门店、多角色、多终端**的健身房数字化管理系统，实现会员服务、门店管理、品牌管理与总部运营的一体化，满足公司业务快速扩展和系统统一化的需求。

公司经营多个健身品牌，每个品牌在产品和价格体系上独立运营，且拥有不同数量的门店（场馆），门店数量会随业务发展增减变化。
在管理模式上，门店独立运营，但财务和数据需按**门店 → 品牌 → 总部**三级汇总。

---

### **2. 产品目标**

1. 提供统一的会员端小程序，覆盖所有品牌的会员服务功能。
2. 提供门店端（小程序简化版）和 Web 门店端（完整功能版），满足教练、前台、门店管理者、品牌管理者的日常运营需求。
3. 提供总部端 Web 系统，实现品牌与门店的全局管理、财务、人事与数据分析。
4. 系统支持**多品牌多门店架构**，可灵活增删品牌和门店。

---

### **3. 系统用户角色与终端**

| 角色      | 使用终端          | 主要功能                       |
| ------- | ------------- | -------------------------- |
| 会员      | 小程序会员端        | 预约、签到、评价、体征记录、查看课程与教练、消费记录 |
| 教练 / 前台 | 小程序门店端（简版）    | 添加会员、办理/续费卡、代理预约、签到管理      |

| 品牌管理者   | Web 品牌端（完整功能） | 管理品牌旗下所有门店                 |
| 总部管理者   | Web 总部端       | 品牌与门店管理、全局财务、人事与数据分析       |

### **6. UI设计规范**

#### **小程序端设计风格**
* **毛玻璃悬浮效果**：
  * 采用半透明毛玻璃材质（backdrop-filter: blur()）
  * 卡片元素悬浮设计，增强层次感
  * 背景模糊效果，突出前景内容

* **精美简约风格**：
  * 极简主义设计理念，去除冗余元素
  * 大量留白，提升视觉舒适度
  * 圆角设计，柔和视觉体验
  * 渐变色彩搭配，增强现代感

* **交互体验**：
  * 微动效设计，提升操作反馈
  * 悬浮按钮和卡片阴影效果
  * 流畅的页面切换动画
  * 响应式触摸反馈

#### **色彩规范**
* **主色调**：现代蓝色系（#007AFF）
* **辅助色**：渐变色彩（蓝紫渐变）
* **背景色**：浅灰白色系（#F8F9FA）
* **文字色**：深灰色系（#333333, #666666）
* **毛玻璃透明度**：80-90%透明度

### **8. 详细功能需求分析**

基于对健身房门店实际运营场景的深入分析，以下是各角色对门店端小程序的详细功能诉求：

#### **8.1 🏋️‍♂️ 私人健身教练（1对1私教）**

**工作场景分析**：私教需要为会员提供个性化的训练指导，建立长期的教练-会员关系，注重训练效果和会员满意度。

**核心功能诉求**：

📅 **课程管理**
- 查看个人课表（今日/本周/本月课程安排）
- 课程详情（会员信息、训练目标、课程时长）
- 课程状态管理（已完成/进行中/已取消）
- 调课申请和请假管理

👤 **会员档案管理**
- 会员基础信息（年龄、性别、联系方式）
- 健康状况（既往病史、运动禁忌、身体评估）
- 训练目标（减脂/增肌/塑形/康复等）
- 身体数据追踪（体重、体脂率、围度等）
- 历史训练记录查看

📝 **训练记录系统**
- 实时记录训练数据（动作、重量、组数、次数）
- 上传训练照片/视频作为记录
- 记录会员训练状态和反馈
- 生成训练报告和进度分析

💬 **会员沟通**
- 与会员即时聊天
- 发送训练建议和饮食指导
- 课后反馈和下次训练安排
- 训练视频和动作指导分享

💰 **收入统计**
- 个人课时费统计
- 月度/季度收入报表
- 提成和奖金查看
- 课程完成率统计

#### **8.2 🧘‍♀️ 团课教练**

**工作场景分析**：团课教练面向群体授课，注重课程氛围和整体效果，需要管理多名会员同时上课。

**核心功能诉求**：

📋 **团课管理**
- 查看团课课表（瑜伽、动感单车、普拉提等）
- 课程报名情况和人数统计
- 教室和设备安排查看
- 课程容量管理（人数上限控制）

✅ **签到管理**
- 快速批量签到功能
- 查看报名会员名单
- 缺席会员统计
- 临时加入会员处理

📚 **教学资源**
- 上传课程音乐和背景音效
- 分享动作教学视频
- 课程流程和动作指导
- 教学素材库管理

📊 **课程反馈**
- 收集会员课后评价
- 课程满意度统计
- 改进建议收集
- 课程质量分析

🎯 **绩效统计**
- 团课收入统计
- 课程评分和排名
- 会员参与度分析
- 课程受欢迎程度

#### **8.3 🏪 前台工作人员**

**工作场景分析**：前台是会员接触最频繁的岗位，需要高效处理大量重复性操作，提供优质的服务体验。

**核心功能诉求**：

🚪 **门禁签到系统**
- 快速扫码签到/签退
- 会员权限验证（会员卡状态、有效期）
- 访客登记和临时权限
- 进出记录统计

👥 **会员服务**
- 快速查询会员信息
- 办理新会员入会手续
- 会员卡充值和续费操作
- 会员卡挂失和补办

💳 **财务操作**
- 收银结算系统
- 开具发票和收据
- 交易记录查询
- 日结和交班报表

❓ **咨询服务**
- 查看课程安排和教练信息
- 健身房设施介绍
- 会员权益说明
- 价格和套餐查询

🔧 **问题处理**
- 记录会员投诉和建议
- 上报设备故障
- 紧急情况处理流程
- 客服工单管理

📈 **数据统计**
- 每日签到人数统计
- 收入流水记录
- 会员流量分析
- 服务质量评估

#### **8.4 👔 门店管理人员**

**工作场景分析**：管理人员需要全局掌控门店运营，基于数据做出决策，确保门店盈利和服务质量。

**核心功能诉求**：

📊 **运营监控仪表板**
- 实时门店运营数据概览
- 关键指标监控（收入、客流、转化率）
- 异常情况预警提醒
- 多维度数据对比分析

👨‍💼 **员工管理**
- 员工工作状态监控
- 绩效考核和评估
- 排班管理和调整
- 培训计划和进度

💼 **财务分析**
- 收入支出详细分析
- 利润率和成本控制
- 预算执行情况
- 财务趋势预测

📈 **会员分析**
- 会员增长和流失趋势
- 会员价值分析（LTV）
- 会员行为分析
- 续费率和活跃度统计

🎯 **营销管理**
- 促销活动效果分析
- 销售转化漏斗
- 市场竞争分析
- ROI计算和优化

⭐ **服务质量监控**
- 客户满意度统计
- 投诉处理跟踪
- 服务评价分析
- 改进措施制定

📋 **决策支持**
- 自定义报表生成
- 数据导出功能
- 趋势分析和预测
- 经营建议推荐

#### **8.5 🔄 角色间协作需求**

**信息共享**
- 会员信息在各角色间实时同步
- 课程安排和变更及时通知
- 异常情况快速传达

**工作流协作**
- 前台办卡后自动分配教练
- 教练请假后自动通知相关会员
- 管理层决策快速传达执行

**权限管理**
- 不同角色看到不同的功能模块
- 敏感信息按权限分级显示
- 操作日志记录和审计
